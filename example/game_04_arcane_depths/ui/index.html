Arcane Depths

一个由 Excel 配置驱动的“战术 Roguelike + 基地经营”示例。通过 tables 插件流水线生成 schema、转换结果与 jsonx 协议，快速搭建可体验的游戏 MVP。

● 多层地牢 & 章节被动 ● Excel → tables → React ● jsonx 协议头输出

Quick Start

  1. 1. 运行 node _rebuild_data.js
  2. 2. 运行 node serialize.js
  3. 3. 使用静态服务器打开 ui/index.html
  4. 4. 借助 Chrome Dev MCP 调试，扩展玩法

当前页面直接消费 /out 下的 JSON / JSONX 产物。
